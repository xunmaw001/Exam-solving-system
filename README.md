# Exam-solving-system
这款基于微信小程序的刷题系统，是备考提分的高效工具。无需安装，微信内一键开启刷题之旅。它涵盖考研、考公、职业资格证等海量题库，题型丰富，答案解析详尽。智能组卷、专项训练、模拟考试多种刷题模式任选，满足不同学习需求。系统依据答题情况生成错题本，精准查漏补缺，还能记录学习进度、分析薄弱点。每日推送精选题目，助你利用碎片化时间学习。无论何时何地，轻松刷题，助力提升应试能力 。
